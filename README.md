# Parallel Agent Leaderboard

## Introduction

This repository is a global leaderboard and benchmarking space for Cursor's parallel agents feature. We believe this technology, while still maturing, has immense potential—I've personally seen it boost my productivity by 2x in daily work. Yet, I also feel it's not fully there yet.

Meanwhile, innovators in other countries are using the same tools to create breakthroughs. This space is for comparing our know-how, building fair benchmarking systems, and continuously improving them. Here, you can see how you rank globally, fostering positive competition that drives mutual development.

## The Shock

What shocked me the most is that my primary communication tool is no longer Slack—it's now Cursor, thanks to the parallel agents feature.

In the past, we assigned tasks to people via Slack. Now, we're assigning them to agents.

If ChatGPT brought 30% or 2x productivity, this could deliver at least 50x. That's the level of breakthrough we're talking about.

## The Difference

This will create a difference comparable to the iPhone's launch. In the future, people will carry personal LLM workstations—like DGX Spark— just as they carry laptops today.

That's the magnitude of change we're facing.
